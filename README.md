# 薛之谦 · 歌曲淘汰赛

一个无需后端的个人歌曲淘汰赛网页。候选曲目按照提供截图的顺序进行首轮配对，投票进度保存在浏览器本地。

## 功能

- 128 首曲目，7 轮单败淘汰
- 手机与桌面自适应
- 本地保存进度与撤回上一票
- 最终生成包含全部历史晋级记录的 PNG 长图
- 可选的本地环境音效，不会自动播放

## 本地预览

```bash
python3 -m http.server 4173
```

访问 `http://localhost:4173`。

## 部署

项目为纯静态文件，可直接部署到 GitHub Pages。Pages 的 Source 请选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/(root)`。
